# README

## 概要
このアプリケーションはお城に少しでも興味がある人を対象にしたアプリケーションです。
あまり城に興味がない方でも投稿した人のステキなお城の画像を見て、行ってみたい城や好きな城が見つかります。
<br>**"あなたの好きな城が見つかる"**<br>
をコンセプトにしています。

**このアプリケーションは現在(2021/2/23)ベータ版です。今後機能やビューを充実させていきます。**
